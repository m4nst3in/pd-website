Sobre o Projeto

O Platform Destroyer é uma plataforma dedicada a fornecer scripts para automatização de atividades em plataformas educacionais. Nossa missão é facilitar o processo através de soluções tecnológicas eficientes. Apenas mantemos os links dos scripts, não nos responsabilizamos pelo mal uso de tais ou quaisquer problemas judiciais.

  • Scripts testados e funcionais
  • Atualizações constantes
  • Suporte via Discord
  • Compatível com principais navegadores
