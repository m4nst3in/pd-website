Como Instalar

  1. Utilize um navegador compátivel, como o Chrome e o Firefox
  2. Clique no botão de download abaixo
  3. Obtenha o script através de nosso servidor de Discord
  4. Copie o script na página do GitHub
  5. Favorite uma página, edite o link deste favorito e cole o script no link
  6. Após isso, siga as instruções da seção Como Usar
  7. Caso não consiga, confira o vídeo ao lado

Tutorial em Vídeo

Como Usar

  1. Faça login na sua conta do Khan Academy
  2. Navegue até o menu inicial do Khan Academy
  3. Execute o script através do bookmarklet
  4. Realize o login no script pelo Discord
  5. Abra o menu e aproveite o script!

Funcionalidades

  • Resolução automática de exercícios
  • Preenchimento de respostas
  • Progresso automático entre questões
  • Compatível com todas as matérias
  • Seguro, anti-banimentos
  • 100% gratuito e atualizado
