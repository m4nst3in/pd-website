TarefaSP Script

Como Instalar

  1. Instale a extensão Tampermonkey no seu navegador
  2. Clique no botão de download abaixo
  3. O Tampermonkey abrirá automaticamente para instalação
  4. Clique em "Instalar" no Tampermonkey

Como Usar

  1. Acesse o portal TarefaSP (Sala do Futuro)
  2. Entre em qualquer atividade disponível
  3. O script será ativado automaticamente
  4. Use os atalhos para controle:
Alt + T Iniciar/Pausar automação
Alt + E Parar execução

Funcionalidades

  • Resolução automática de todas as atividades
  • Preenchimento inteligente de respostas
  • Avanço automático entre questões
  • Suporte a todos os tipos de exercícios

Avisos Importantes

  • Use com intervalos regulares
  • Evite múltiplas sessões simultâneas
  • Mantenha o script sempre atualizado
