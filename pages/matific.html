Matific Script

Como Instalar

  1. Instale a extensão Tampermonkey no seu navegador
  2. Clique no botão de download abaixo
  3. O Tampermonkey abrirá automaticamente para instalação
  4. Clique em "Instalar" no Tampermonkey

Como Usar

  1. Acesse sua conta no Matific
  2. Entre em qualquer atividade matemática
  3. O script será ativado automaticamente
  4. Use os atalhos para controle:
Alt + M Iniciar/Pausar automação
Alt + Q Parar execução

Funcionalidades

  • Resolução automática de problemas matemáticos
  • Cálculos precisos e rápidos
  • Progresso automático entre exercícios
  • Compatível com todos os níveis

Avisos Importantes

  • Utilize com moderação
  • Faça pausas entre as sessões
  • Mantenha o script atualizado
