Como Instalar

  1. Instale a extensão Tampermonkey no seu navegador
  2. Clique no botão de download abaixo
  3. O Tampermonkey abrirá automaticamente para instalação
  4. Clique em "Instalar" no Tampermonkey

Tutorial em Vídeo

Como Usar

  1. Faça login na sua conta do KhanAcademy
  2. Navegue até a atividade desejada
  3. O script será ativado automaticamente
  4. Use os atalhos de teclado:
Alt + A Ativar/Desativar automação
Alt + S Parar execução

Funcionalidades

  • Resolução automática de exercícios
  • Preenchimento de respostas
  • Progresso automático entre questões
  • Compatível com todas as matérias

Avisos Importantes

  • Use com moderação
  • Evite múltiplas execuções simultâneas
  • Mantenha o script atualizado
