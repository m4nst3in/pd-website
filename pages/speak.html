SPeak Script

Como Instalar

  1. Instale a extensão Tampermonkey no seu navegador
  2. Clique no botão de download abaixo
  3. O Tampermonkey abrirá automaticamente para instalação
  4. Clique em "Instalar" no Tampermonkey

Como Usar

  1. Entre na sua conta do SPeak
  2. Acesse qualquer atividade
  3. O script será iniciado automaticamente
  4. Use os atalhos para controle:
Alt + R Iniciar/Pausar resolução
Alt + X Parar script

Funcionalidades

  • Resolução automática de exercícios
  • Simulação de voz para atividades orais
  • Preenchimento de respostas escritas
  • Progresso automático entre lições

Avisos Importantes

  • Use com intervalos entre as atividades
  • Evite usar em horários de pico
  • Atualize o script regularmente
